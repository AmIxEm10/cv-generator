# 📄 CV Generator

Générateur de CV moderne, gratuit et entièrement côté client (no server required).

## ✨ Fonctionnalités (roadmap)

- [x] Maquette visuelle A4 – design pro à deux colonnes
- [x] Formulaire interactif avec aperçu en temps réel
- [x] Export PDF haute qualité via html2pdf.js
- [x] 6 thèmes de couleurs
- [x] 4 combinaisons de polices

## Full Stack

- **HTML5 / CSS3 vanilla** – zéro dépendance backend
- **Google Fonts** – DM Sans + Playfair Display
- **html2pdf.js** – export PDF

## 🚀 Lancer le projet

```bash
# Cloner le repo
git clone https://github.com/AmIxEm10/cv-generator.git
cd cv-generator

# Ouvrir directement dans le navigateur
open index.html
```

## 📁 Structure

```
cv-generator/
├── index.html        # Application complète
1└── README.md
```

## 🖨️ Export PDF

L'export est intégré via `html2pdf.js`. Cliquer sur **Exporter PDF** dans l'application.
